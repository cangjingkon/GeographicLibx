NOTE: It's not necessary to download these files to use GeographicLib.

GeoidHeights.dat.gz contains 500000 points together with the geoid
heights accoring to various earth gravity models.  Each line of the file
contains:

  * latitude (degrees, exact)
  * longitude (degrees, exact)
  * EGM84 height (meters, accurate to 1 mm)
  * EGM96 height (meters, accurate to 0.1 mm)
  * EGM2008 height (meters, accurate to 1 mm)

For more information, see

 http://geographiclib.sourceforge.net/html/geoid.html#testgeoid
